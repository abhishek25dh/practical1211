# practical1211

Simple Python animation generator for a walking stick figure.

## Usage

1. Install dependencies:

```bash
pip install matplotlib pillow
```

2. Generate an animation (GIF):

```bash
python3 stickfigure_animation.py --output output/stickfigure.gif
```

3. Generate MP4 (requires ffmpeg installed on your system):

```bash
python3 stickfigure_animation.py --output output/stickfigure.mp4
```
